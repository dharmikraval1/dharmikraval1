## 🛠️ Featured Project

### 📄 [AI Document Creator](https://github.com/dharmikraval1/ai-document-creator)

> **Ask your AI assistant to document any codebase — get per-file docs, a polished README, and architecture diagrams, delivered as files or a pull request.**

- 🔌 Works as an **MCP server** inside Claude Code, Claude Desktop, Cursor, Codex & any MCP host
- 🐍 Published on **PyPI** (`pip install ai-doc-creator`) with a full CLI
- ⚙️ Ships as a **GitHub Action** — docs regenerate on every push and open a PR
- 🧩 Supports Anthropic, OpenAI, Azure OpenAI, AWS Bedrock & Ollama — or **zero API key** via MCP sampling
- 🛡️ Hardened with SSRF guards, sandboxing, rate limits & CI, plus incremental hash-based caching
- 📊 Generates **Mermaid architecture diagrams** from real imports, so they're always accurate

```bash
claude mcp add ai-doc-creator -- uvx --from ai-doc-creator ai-doc-creator-mcp
```
